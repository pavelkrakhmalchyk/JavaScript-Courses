JavaScript, often abbreviated as JS, is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is a language that is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm.

JavaScript is officially managed by Mozilla Foundation, and new language features are added periodically. However, only some JavaScript engines support these new features:

  1. property getter and setter functions (supported by WebKit, Gecko, Opera,[50] ActionScript, and Rhino)
  2. conditional catch clauses
  3. iterator protocol (adopted from Python)
  4. shallow generators-coroutines (adopted from Python)
  5. array comprehensions and generator expressions (adopted from Python)
  6. proper block scope via the let keyword
  7. array and object destructuring (limited form of pattern matching)
  8. concise function expressions (function(args) expr)
  9. ECMAScript for XML (E4X), an extension that adds native XML support to ECMAScript (unsupported in Firefox since version 21)
